The compile.bat file needs to be added into folder name “processing-4.0.1”. 
This is the folder wherein processing-java.exe and processing.exe files are being present. 
In order to use this compile.bat file, change the input path present in this file to the folder path which contains the processing files that are needed to be converted to java. 
And then set the output path to the one wherein the converted java files to be present in. 

Then to run this file in the command prompt just type the command as - run compile.bat. This will help in exporting processing file to java file.
